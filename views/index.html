This is Tipical...

Tipical is a simple anonymous tips app. Search for your city using the far-right text box. If someone has left a tip, it will show up!

Click on the map to drop a pin, then give it a tip using upper left text box.

Tipical was created as a learning exercise early in the Thinkful coding bootcamp for Node, Mongo/Mongoose, and OpenLayers. It was made by Zachary Williams and Thien Pham.
